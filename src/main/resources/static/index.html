Knowledge Base API Documentation

Query Endpoint

Use this endpoint to query facts from the knowledge base.

POST /query
Content-Type: application/json

{
  "query": "What is the capital of France?"
}

Store Endpoint

Use this endpoint to store new facts in the knowledge base.

POST /store
Content-Type: application/json

"The capital of France is Paris."

Buildship Integration

Endpoints for Buildship integration:

POST /buildship/query
Content-Type: application/json

{
  "question": "What is the capital of France?",
  "sessionId": "user123"
}
    
POST /buildship/store
Content-Type: application/json

{
  "question": "The capital of France is Paris.",
  "sessionId": "user123"
}
